What is Seismic?

                
Seismic is an encrypted blockchain that allows for confidential transactions and smart contracts. It uses specialized encrypted data types:

                
    
                    
  • suint/sint - Encrypted integers
    • 
                    
  • saddress - Encrypted Ethereum addresses
    • 
                    
  • sbool - Encrypted boolean values
    • 
                

                
When you use these types in your smart contracts, Seismic automatically handles the encryption and decryption without exposing sensitive data on-chain.

                
                
Key Benefits

                
    
                    
  • Private financial transactions
    • 
                    
  • Confidential business logic
    • 
                    
  • Protection of user identity
    •
